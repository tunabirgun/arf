The Library

Every reference you cite, kept in full, and never allowed to rot.

Arf is not a reference manager competing with Zotero, and it does not ask you to abandon one. But a research-notes tool needs to hold the sources those notes are about, in detail, and to make citing them effortless. That is the Library: a browsable record of every reference in your work — journal articles, books, magazine and newspaper pieces, chapters, conference papers, preprints, datasets, theses, and web pages.

Fetched from open bibliographic sources

You add a reference by giving Arf an identifier — a DOI, an ISBN, or an arXiv ID — and it fills in the title, authors, year, and venue for you from open, no-account sources:

You haveArf fetches from
A DOICrossref — the canonical scholarly record
An ISBNOpen Library
An arXiv IDCrossref, via the paper's arXiv DOI

Only the identifier you typed ever leaves your device — never your notes. Each reference records which source it came from, and you can correct any field by hand. If a lookup finds nothing, you add the reference and fill the details yourself.

Web pages that don't disappear

A web citation is worthless once the page changes or vanishes. When you add a web source, Arf preserves it through the Internet Archive's Wayback Machine — capturing a snapshot and recording the archived link, the capture date, and the date you accessed it. The citation points at something permanent.

Cite as you write, export anywhere

Insert a citation without leaving the editor: the @ button opens a picker over your library and drops a [@citekey] at the cursor. In the reading view it renders as Author Year and jumps to the reference on click. Two works by one author in the same year are told apart automatically — Author 1981a, Author 1981b — and you can switch on an end-of-note bibliography in Settings, in the citation style you choose. Organize the library itself into folders, by dragging a reference onto one or typing a path. When it is time to leave — to a manuscript, a co-author, or another tool — the Library exports to every format you are likely to need:

Managers

BibTeX · RIS

RIS is what EndNote, Zotero, and Mendeley read. BibTeX for LaTeX workflows.

Interchange

CSL-JSON

The format Arf stores internally and the lingua franca of citation tools.

Formatted

APA · Nature · &c.

Rendered in any of thousands of citation styles for pasting straight into a document.

Deposit

Zenodo

Deposition metadata for archiving your own work.

Import runs the same way in reverse, so an existing .bib or RIS file comes in cleanly.
